# tg-message-forwarder

Сервис, который принимает сообщения от пользователя в Telegram и пересылает их админу в виде красиво отформатированного JSON.

## Как работает

1. Пользователь отправляет сообщение боту в Telegram.
2. Бот обрабатывает сообщение и сериализует его в JSON со структурой:

```json 
{
    "id": "e1b5e4d8-8c9f-4f5d-9a21-123456789abc",
    "telegram_id": 123456789,
    "telegram_username": "some_username",
    "text": "Привет, это тестовое сообщение",
    "created_at": "2025-11-23T18:25:43.511Z"
}
```

3. Сообщение публикуется в Kafka.
4. Консьюмер читает сообщения из Kafka и отправляет их администратору в Telegram, форматируя сообщение в виде блока кода Markdown с подсветкой JSON.
